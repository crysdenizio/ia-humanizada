# PRD — Denizio's IA Global Solutions

## Original Problem Statement
Site premium, moderno e altamente persuasivo para "Denizio's IA Global Solutions" (automação inteligente).
Estilo futurista premium (Apple + Tesla), preto profundo (#000000) com dourado metálico (#FFD700, #B8860B).
Objetivo: gerar leads para clínicas, consultórios, empresas (atendimento WhatsApp, IA humanizada, automação 24h).

## Architecture
- **Stack**: React 19 + Tailwind + Framer Motion + Lucide React (frontend-only, sem backend)
- **Tipografia**: Outfit (headings) + Inter (body)
- **Páginas**: Single-page landing com 10 seções
